# Hugo

Статический генератор сайта это скрипт, который из различных файлов (картинок, Markdown разметки и т.п) генерирует статический HTML-сайт.

Hugo — один из популярных генераторов статических сайтов с открытым исходным кодом, написан на языке Go. 

## [Офицальный сайт](https://gohugo.io/)

> Hugo is one of the most popular open-source static site generators. With its amazing speed and flexibility, Hugo makes building websites fun again.

## Плюсы: 

- Поддержка markdown разметки
- Хорошая SEO оптимизация
- Простота использования (следуя инструкции с офф.сайта, можно создать свой проект буквально за 5 минут)
- Простота в деплое. Нам по факту надо хостить статические html + css + картинки. Можно разместить на бесплатном github pages или же арендовать хостинг с доменом, поставить nginx и раздавать им
- Много готовых тем и шаблонов

## Минусы:

- Придется заново передеплоивать сайт, после того как изменили или добавили новую статью
- Нет бекенда и сделать можно далеко не все

## Для чего хорош

- Hugo хорош для экспертных блогов (тут удобная подсветка синтаксиса при встраивании кода и в целом я люблю markdowm)
- Для не программистов. Не надо писать код и разбираться в нем. Почитай [синтаксис markdown](https://www.markdownguide.org/basic-syntax/), установи по инструкции и начинай писать
- Для сайтов портфолио - идеальный вариант. Можно разместить свои проекты и ссылки на другие соц.сети
